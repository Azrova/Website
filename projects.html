AzrovaDash

TypeScript Bun Dashboard

AzrovaDash is a sleek, next-gen client dashboard for the Pterodactyl Panel, built with TypeScript, powered by Bun, and designed for speed, customization, and modern aesthetics.

Features

  • Lightning-fast performance with Bun runtime
  • Enhanced server management and creation capabilitie
  • Advanced server statistics and monitoring
  • Responsive design for all devices
  • Fully open-source
