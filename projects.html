AzrovaDash

A sleek, next-gen client dashboard for the Pterodactyl Panel, built with TypeScript, powered by Bun, and designed for speed, customization, and modern aesthetics.

TypeScript Bun Tailwind CSS

FEATURES

  • Ultra-modern UI with sleek dark mode design
  • Enhanced server management and creation capabilities
  • Real-time server statistics and monitoring
  • Responsive design for all devices
  • Enhanced console with improved functionality

TECHNICAL DETAILS

  • Built with TypeScript for type safety and better code quality
  • Powered by Bun for superior performance and speed
  • Designed with Tailwind CSS for consistent styling
  • RESTful API integration with Pterodactyl Panel
  • Open source under Apache license
