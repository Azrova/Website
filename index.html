WHO WE ARE

AZROVA is a collective of developers dedicated to creating cutting-edge open source projects that push the boundaries of what's possible. We focus on building tools that are not just functional, but exceptionally designed and optimized for performance.

Our projects span from game server management dashboards to upcoming developer tools, all built with modern technologies and a focus on user experience. We believe in the power of open source to drive innovation and make powerful tools accessible to everyone.

Currently, we're focusing on modernizing tools in the game server management space, with our flagship project AzrovaDash revolutionizing how server administrators interact with the Pterodactyl Panel.

FAQ

Are your projects 100% open source?

Absolutely. All AZROVA projects are released under open source licenses. We believe in transparency and community collaboration. You can find all our source code on our GitHub repositories.

Can I contribute to AZROVA projects?

Yes! We welcome contributions from the community. Whether it's code, documentation, design, or ideas, feel free to open issues or pull requests on our GitHub repositories.

What technologies do you use?

We use modern technology stacks including TypeScript, Bun, Tailwind CSS, and other cutting-edge tools. Each project may have its specific technology choices based on requirements, but we always prioritize performance, security, and developer experience.

What's next for AZROVA?

We're constantly working on new projects and improving existing ones. Our roadmap includes more tools for game server management, developer productivity, and infrastructure automation. Stay tuned by following us on GitHub.
